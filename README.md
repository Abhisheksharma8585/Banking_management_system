# 🏦 Banking Management System

A robust **Java-based Banking Management System** developed using **Object-Oriented Programming (OOP)** principles. The application enables users to create bank accounts, perform transactions, and manage customer banking operations through a simple console interface.

---

## 📌 Overview

The Banking Management System simulates core banking functionalities such as account creation, deposits, withdrawals, balance inquiry, and transaction history. The project demonstrates practical implementation of Java OOP concepts including inheritance, encapsulation, abstraction, and polymorphism.

---

## ✨ Features

- 👤 Create New Bank Account
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Account Balance
- 🧾 View Transaction History
- 🏦 Support for Savings & Checking Accounts
- 🔒 Input Validation
- ⚡ Console-based Interactive Menu
- 🧩 Modular Code Structure
- 📂 Package-based Organization
